
아파치 톰캣(Apache Tomcat)

자바 기반의 웹 애플리케이션을 실행하기 위한 **웹 컨테이너(Web Container)**입니다. 
웹 서버의 한 종류이기도 하지만, 정적인 HTML 페이지를 제공하는 일반적인 웹 서버와는 다르게, 
동적인 자바 기반의 웹 콘텐츠를 처리하는 데 특화되어 있습니다.


주요 기능
- 서블릿(Servlet)과 JSP(JavaServer Pages) 실행: 
  톰캣은 서블릿과 JSP를 실행하여 동적인 웹 페이지를 생성하고 사용자 요청에 응답하는 기능을 수행합니다. 
  서블릿은 서버 측에서 실행되는 자바 프로그램이며, JSP는 HTML 안에 자바 코드를 포함시켜 동적으로 웹 페이지를 생성하는 기술입니다.

- HTTP 서버 역할: 
  클라이언트(웹 브라우저)의 HTTP 요청을 받아들이고 처리하여 결과를 다시 클라이언트에게 보내주는 웹 서버 역할도 합니다.

- WAS(Web Application Server) 역할: 
  톰캣은 웹 애플리케이션을 실행하는 데 필요한 환경을 제공하기 때문에 경량 WAS로 분류되기도 합니다. 
  완전한 WAS보다는 기능이 적지만, 대부분의 웹 애플리케이션을 운영하기에는 충분합니다.
